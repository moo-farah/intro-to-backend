# Intro to Backend Web Development

Welcome to my backend web development learning journey! This repository contains the projects and exercises I am building as I learn how to create secure, scalable, and efficient server-side applications.

*   **JavaScript:** The core programming language used for both logic and development.
*   **Node.js:** The runtime environment that allows me to run JavaScript on the server.
*   **Express.js:** The web framework used to handle routing, middleware, and HTTP requests easily.
*   **MongoDB:** A NoSQL database used to store and manage application data flexibly.
*   **Postman:** The API client I use to test endpoints, send HTTP requests, and verify responses.
